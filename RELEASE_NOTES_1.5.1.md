# Mineradio v1.5.1

本次更新增强 Wallpaper Engine 导入、第三方音源容错、本地音频兼容和严格单曲循环。

## 主要更新

- 支持批量导入、保存和切换多个 LX 兼容音源。
- 在线歌曲无法播放时自动尝试其他已导入音源。
- 修复各平台搜索结果点击播放无反应。
- 搜索后播放不会清空关键词和结果。
- 搜索歌曲可加入自建歌单；自建歌单会显示在主页。
- 支持搜索当前歌单和跨全部歌单搜索，兼容中文输入法。
- 本地歌曲缺少翻译时自动补查并合并翻译歌词。
- 改进普通小枸、小枸概念版、咪咕及其他公开歌单导入。
- 修复平台返回短页时歌单提前结束；QQ 电脑分享链接支持新旧接口并行回退，小蜗和小菇按空页、无新增或声明总数判断结束。
- 小枸分享页不再把十几首预览歌曲直接当作完整歌单，无法取得全量数据时会明确提示而不是保存残缺结果。
- Wallpaper Engine 图片和视频使用原始媒体导入，Scene 使用静态预览；不会修改 Windows 桌面壁纸。
- 壁纸列表采用三列 16:9 卡片，支持多分辨率、高清预览、去重、隐藏和恢复。
- 第三方平台遇到 429、502、超时等临时错误时自动重试。
- 音源支持指数退避、`Retry-After`、失败冷却、短期地址缓存与自动换源。
- 本地音频支持文件头检查、非破坏性修复副本，以及可选 FFmpeg WAV 回退。
- 识别 NCM、QMC、KGM、MFLAC、MGG 等加密格式，只提示，不破解。
- 本地和在线歌曲支持严格单曲循环，手动或托盘切歌时重新播放当前歌曲。
- 修复桌面歌词三行、四行被强制降级为双行的问题；现在会真实显示对应句数，并分层显示翻译和当前句进度。
- 桌面歌词长句改为固定居中缩放，不再持续向左滚动；当前播放句的原文和翻译会同时高亮发光。
- 修复桌面端点击歌单内容后面板意外收回。
- 优化 3D 渲染、长列表、播放进度刷新和后台资源占用。
- Electron 更新至 39.8.10。

## 安装说明

- Windows x64。
- 可直接覆盖安装 v1.4.x 或 v1.5.0，现有设置、自建歌单和音源配置会保留。
- 安装包未使用商业代码签名，Windows 首次运行时可能显示安全提醒。

## 使用限制

- 在线播放需要用户自行配置合法可用的 LX 兼容音源。
- 第三方平台接口、访问限制或分享链接变化可能导致搜索或导入暂时失败。
- Wallpaper Engine 原生 Scene `.pkg` 无法脱离 Wallpaper Engine 在 MR 内还原；MR 不会调用它修改 Windows 桌面壁纸。
- 只有头部异常或浏览器不兼容的普通音频可创建修复/转换副本；严重损坏或平台加密音频无法修复，MR 会跳过并提示。

# Mineradio 1.5.3 补充版

## 下载

- [Mineradio.Setup.1.5.3.exe](https://github.com/ww085213/Mineradio-LX-Music/releases/download/v1.5.3-supplement/Mineradio.Setup.1.5.3.exe)
- [GitHub Release](https://github.com/ww085213/Mineradio-LX-Music/releases/tag/v1.5.3-supplement)

## 更新内容

- 新增音源“一键导入”：可以一次选择多个本地落雪兼容音源 `.js` 文件并批量导入。
- 未配置音源时，点击音源状态会直接打开批量音源文件选择。
- 导入过程逐个验证音源，成功后刷新当前音源状态。
- 保留检查更新能力，后续发布 `1.5.4` 或更高版本时可直接在软件内更新。

## 说明

这是 `1.5.3` 的补充安装包，应用版本号仍保持 `1.5.3`。因此已经安装普通 `1.5.3` 的用户不会通过“检查更新”发现它，需要手动下载补充版安装包安装。

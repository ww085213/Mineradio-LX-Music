# Mineradio 1.5.5

## 重点更新

- 支持在安装向导中选择安装路径，不再双击后直接安装。
- 场景录制升级到 60 FPS H.264，完整携带 FFmpeg 与 RePKG。
- 恢复 GIF 预览、播放条频谱和播放状态动画。
- 修复桌面融合的鼠标交互、操作层快捷键、退出后窗口尺寸恢复与任务栏图标。
- 修复音频输出列表匿名或缺失，现在可显示并切换 Windows 的真实扬声器、耳机和虚拟音频设备。
- 完善音源、歌词与跨平台歌单导入，修复大歌单解析和歌曲信息缺失。
- 优化缓存、后台节流、录制流程和运行依赖占用。
- 加强安装目录标记、卸载保护和快捷方式清理逻辑。

## 安装

下载 `Mineradio.Setup.1.5.5.exe`，在安装向导中选择一个独立的 Mineradio 文件夹。已有版本升级前请先退出 Mineradio。

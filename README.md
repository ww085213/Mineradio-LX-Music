# Mineradio - LX Music（落雪音乐）/ MR

> **项目来源与署名：** 本仓库基于原作者 **[@XxHuberrr](https://github.com/XxHuberrr)** 的开源项目 **[XxHuberrr/Mineradio](https://github.com/XxHuberrr/Mineradio)** 进行非官方二次开发。感谢原作者及原项目所有贡献者。本仓库保留原项目版权与 MIT License 声明，不主张替代原作者身份，也不代表原作者为本版本提供背书、维护、授权或担保。

MR 是一个以**本地音乐播放、歌词展示和音乐可视化**为核心的桌面应用。用户可以扫描并播放自己有权使用的本地音频文件，不需要安装 LX Music，也不需要登录任何第三方音乐平台账号。

项目同时提供若干可选扩展能力，例如 LX Music 本机联动、`.lxmc` 歌单导入、歌单元数据导入、歌词补查，以及兼容用户自行配置的数据源。上述扩展能力并非本地播放所必需，其可用性、合法性和稳定性取决于用户配置及第三方服务。

## GitHub 项目描述

MR 是一个以本地音乐库、歌词舞台、桌面歌词和 3D 音乐可视化为核心的桌面播放器。其本地播放能力可独立运行；LX Music 本机联动、歌单导入和用户自配数据源属于可选扩展功能。

## 核心定位

- 本地音乐播放器
- 本地音乐库与自建歌单管理
- 歌词舞台与桌面歌词
- 3D 歌单架和音乐可视化
- 可选的 LX Music 本机联动
- 可选的歌单元数据导入
- 用户自行配置的数据源兼容框架

## 支持的内容来源

### 本地内容

- MP3
- FLAC
- WAV
- M4A
- OGG
- 本地文件夹音乐库
- 本地自建歌单
- 本地歌词与封面

### 可选扩展

- LX Music `.lxmc` 歌单文件导入
- 用户主动开启的 LX Music 本机开放 API 联动
- 用户自行配置的兼容数据源
- 公开分享页面或歌单元数据导入
- 用户自行配置的歌词与翻译歌词补查

> 可选扩展不代表相关第三方平台对本项目进行了授权、合作或背书。

## 主要功能

- 扫描并播放本地音乐文件
- 管理本地音乐库、本地队列和自建歌单
- 读取本地封面与本地歌词
- 显示歌词舞台、桌面歌词和音乐可视化
- 支持本地图片/视频背景与 Wallpaper Engine 背景读取
- 支持可选的 LX Music `.lxmc` 歌单导入和本机联动
- 支持用户自行配置的数据源兼容框架

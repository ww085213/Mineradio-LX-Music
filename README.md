# Mineradio - LX Music（落雪音乐）/ MR

> **项目来源与署名：** 本仓库基于原作者 **[@XxHuberrr](https://github.com/XxHuberrr)** 的开源项目 **[XxHuberrr/Mineradio](https://github.com/XxHuberrr/Mineradio)** 进行非官方二次开发。感谢原作者及原项目所有贡献者。本仓库保留原项目版权与 MIT License 声明，不主张替代原作者身份，也不代表原作者为本版本提供背书、维护、授权或担保。

MR 是一个以**本地音乐播放、歌词展示和音乐可视化**为核心的桌面应用。用户可以扫描并播放自己有权使用的本地音频文件，不需要安装 LX Music，也不需要登录任何第三方音乐平台账号。

项目同时提供若干可选扩展能力，例如 LX Music 本机联动、`.lxmc` 歌单导入、歌单元数据导入、歌词补查，以及兼容用户自行配置的数据源。上述扩展能力并非本地播放所必需，其可用性、合法性和稳定性取决于用户配置及第三方服务。

## Mineradio 1.5.0 新版亮点

1.5.0 主要增强 Wallpaper Engine 导入、自定义数据源容错、本地音频兼容和单曲循环：

- **多数据源管理：** 支持用户导入多个兼容数据源、手动切换，并在网络超时、临时服务错误或解码失败时尝试备用配置。程序不会提供或分发具体数据源。
- **搜索界面：** 修复多个来源搜索结果点击无反应的问题；搜索文字和结果在播放后继续保留。
- **自建歌单：** 支持新建歌单、从搜索结果收藏、主页显示、详情浏览，以及当前歌单和全部歌单的中文搜索。
- **本地歌词：** 本地歌曲缺少歌词翻译时，可按用户配置尝试补查，并缓存原文与翻译。
- **Wallpaper Engine：** 图片和视频使用原始媒体导入；Scene 导入静态预览，不修改 Windows 桌面；支持多分辨率、三列 16:9 列表、高清预览、去重、隐藏与恢复。
- **网络容错：** 对超时、限流和临时服务错误进行有限重试、退避和失败冷却；不会尝试绕过账号验证、付费限制或访问控制。
- **本地音频：** 播放前检查 MP3、FLAC、WAV、OGG、M4A 文件头；异常头部可创建非破坏性修复副本，原生解码失败时可调用 FFmpeg 转换为 WAV。
- **加密格式识别：** 识别 NCM、QMC、KGM、MFLAC、MGG 等平台加密文件并明确提示，不提供解密、破解或绕过保护功能。
- **单曲循环：** 本地歌曲及用户合法数据源返回的可播放内容统一支持循环；换源和回退后保留播放模式。
- **桌面歌词：** 单行、双行、三行、四行会真实控制显示句数；支持上一句、当前句、后续句、翻译分层和当前句进度高亮。
- **性能：** 清理重复旧代码、降低无效 DOM 更新和高刷新率场景中的 GPU 空转，长列表按可见区域绘制。
- **安全：** Electron 升级到 39.8.10；依赖状态以仓库当前审计结果为准。
- **歌单导入：** 新增本地文件、本地文件夹和 `.lxmc` 歌单导入入口，支持一次导入一个或多个 `.lxmc` 文件。
- **歌单元数据导入：** 改进公开分享页面、不同接口格式、分页、去重和完整性判断。该功能只应在用户有权访问相关数据且符合平台规则时使用。
- **来源兼容：** 支持界面中标注的多个来源进行元数据搜索或歌单导入；实际能力取决于用户配置和第三方服务。
- **倍速播放：** 支持 `0.50× ～ 2.00×` 调节。
- **音调控制：** 支持 `-12 ～ +12` 半音调节，可与倍速独立设置并自动保存。
- **歌词补查：** 优化日语、英语、多歌手及 Live、Remix、Version、Feat. 等歌曲匹配。
- **翻译歌词：** 支持读取兼容数据源返回的翻译歌词字段；补查失败时保留原文，并减少把罗马音误判为中文翻译。
- **系统托盘：** 恢复 Windows 托盘图标，支持显示、隐藏和退出；关闭窗口时可隐藏到托盘。
- **窗口行为：** 最小化到任务栏，支持拖动窗口边缘和四角调整大小。
- **首次安装：** 不再依赖开发者电脑中的固定 AppData 路径；缺少用户配置文件时可正常启动。
- **数据源配置：** 优先读取 MR 保存的用户配置，也可在用户明确操作后迁移本机已有配置；未配置时会提示。
- **本地音乐：** 增强 MP3、FLAC、WAV、OGG、M4A 的识别和本地地址读取回退。

> **已知问题：**
>
> - 第三方服务或用户配置的数据源不可用时，在线搜索、歌单导入、歌词补查或在线播放可能失败。
> - 损坏或受保护的音频文件无法保证恢复，程序只会跳过、提示或创建转换副本。
> - Wallpaper Engine 原生 Scene `.pkg` 不能直接嵌入 MR 窗口，MR 不会因此修改系统桌面壁纸。
> - 在线功能需要用户自行配置合法可用的数据源，并自行承担相应授权、服务条款和网络可用性责任。

## 独立播放与可选联动

MR 的本地音乐播放、歌单、歌词、音效和可视化功能可以独立使用，不依赖 LX Music，也不要求登录任何音乐平台账号。

用户主动启用 LX Music 本机联动后，MR 可能读取用户本机 LX Music 开放 API 提供的播放状态、歌单元数据、歌词或控制接口。该联动仅在用户本机进行，用户应自行确认 LX Music 版本、设置及相关许可允许此类使用。

`.lxmc` 文件导入属于可选功能。MR 不会内置或打包 LX Music，也不会提供 LX Music 账号、平台账号、会员权益或第三方访问凭据。

用户自行配置的兼容数据源可能用于搜索元数据、补查歌词或获取可播放地址。维护者不提供、推荐、验证或担保任何具体数据源，也不保证其合法性、稳定性和持续可用性。

## 主要功能

### 本地播放

- 扫描本地文件夹并建立音乐库
- 支持本地歌曲队列播放
- 支持本地歌单收藏和管理
- 支持重复、顺序、随机等播放模式
- 支持本地封面读取
- 支持本地歌词读取
- 支持播放进度、音量、上一首、下一首等基础控制

### LX Music 本机连接（可选）

- 导入 LX Music `.lxmc` 歌单
- 在用户主动开启开放 API 后读取本机歌单列表
- 调用用户本机 LX Music 的本机控制能力
- 同步本机播放状态、标题、歌手、封面、时长和进度
- 同步兼容歌词与翻译歌词
- 对本机联动中的临时状态不同步进行有限重试

> LX Music 本机连接不包含任何音乐平台账号登录，也不应与绕过访问控制的数据源组合使用。

- 支持用户自建歌单收藏和搜索
- 支持网络异常时的有限重试和备用配置切换

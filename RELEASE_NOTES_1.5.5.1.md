# Mineradio 1.5.5.1

## 重点更新

### 全新音乐首页

- 新增首页卡片布局，集中展示继续播放、音乐库、每日推荐和最近播放。
- 新增“今日聆听”统计，可查看聆听时长、今日歌曲数量和连续聆听天数。
- 新增“接下来播放”，可以直接查看并播放队列中的下一首歌曲。
- 新增各平台排行榜和音乐电台入口，音乐发现和场景漫游更直观。
- 每日推荐会结合最近播放和音乐库生成，并减少与前一天推荐内容的重复。

### 全新音乐库与歌单管理

- 新增独立音乐库页面，把本地音乐、在线导入歌单和自建歌单统一管理。
- 支持按歌名、歌手、专辑或歌单搜索。
- 支持导入本地文件夹、导入平台歌单以及新建歌单。
- 歌单详情支持播放全部、自定义封面、重新同步、批量选择、调整歌曲顺序和删除歌单。
- 删除歌单不会删除本地歌曲文件，也不会影响当前播放队列。

### 音乐电台与场景漫游

- 新增 16 种音乐电台，按“为你推荐、场景、风格、能量”分类浏览。
- 包含私人漫游、心动模式、今日漫游、通勤节拍、深夜氛围、专注电台、热门 DJ、高燃模式、游戏战歌、二次元、日本流行、R&B 夜色、华语流行、摇滚现场、BGM 背景音乐和电子脉冲。
- 点击状态卡片即可自动生成连续播放队列；优先从本地音乐和常听偏好中挑选，在用户已配置音源时再通过多平台搜索补足，每次最多生成 60 首。
- 支持按分类筛选以及“换一批顺序”，快速切换不同聆听场景。

### 各平台排行榜

- 新增综合榜及小秋、小芸、小蜗、小狗、小菇 5 个独立平台榜单。
- 综合榜目标 100 首，混合五个平台榜单；榜单缓存每 6 小时更新，也可手动刷新。
- 支持播放全部、直接播放指定歌曲、设为下一首以及收藏到歌单。
- 在用户已导入对应合法音源后，可读取各平台自己的榜单数据。

### 安装与升级修复

- 修复安装包、主程序、卸载程序和快捷方式图标不一致的问题，统一使用 Mineradio 黑色 MR 图标，不再显示 NSIS 默认地球图标。
- 安装向导现在可由用户选择是否创建桌面快捷方式，默认不创建；用户删除桌面快捷方式后，程序不会再次自动生成。
- 加强覆盖安装保护：升级只替换程序文件，不触碰 `%APPDATA%\Mineradio` 中的歌单、设置、音源配置和其他用户数据。
- 完整检查全新用户安装内容，139 个程序文件与当前正式程序一致，FFmpeg、RePKG、可视化和本地播放等核心资源齐全。
- 按安全要求不内置、不自动安装任何第三方音源；需要在线音源功能时，由用户自行导入合法来源。
- 安装包、运行时更新版本和 GitHub 更新仓库统一为 `1.5.5.1` / `ww085213/Mineradio-LX-Music`。
